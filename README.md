# Codex Workspace

Локальный стартовый сайт, который можно запускать прямо здесь и дальше дорабатывать через Codex CLI.

## Запуск

```bash
cd /root/codex-site
npm run dev
```

По умолчанию сайт открывается на `http://127.0.0.1:4173`.

## Структура

- `index.html` — точка входа
- `src/main.js` — разметка страницы
- `src/styles.css` — стили и анимации
